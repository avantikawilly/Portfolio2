Scratch Project

September 3, 2019. CSP
The objective of this project was to design and create your own game or story utilizing Scratch. My partner and I decided to code a Matrix Mini Maze Ball Game. The goal of the game was to get the ball to the end of the maze without hitting obstacles or running out of time. The game consists of several levels that progressively get harder as the player advances. Each level would have a different maze layout, new obstacles for the player to avoid, as well as a set time limit. At the end of each level, your score would be shown on the scoreboard and it would be compared with the highest score you have achieved while playing that game. My role was to both brainstorm ideas for the game as well as code it, my partner and I took turns coding the game and it was challenging to figure out the many different commands we needed to implement for our game to be executed perfectly. In the end, I was able to learn how to think critically to solve problems, as well as learn how to appeal to the users you are targeting with your game.
